# Case De Portfolio

## Resumo

Template multi-tema para negocios de alimentacao, com tres demos publicas: restaurante, pizzaria e hamburgueria. O projeto nasceu como tentativa comercial inicial e hoje foi reposicionado como estudo de produto, frontend tematico e backend/admin experimental.

## Problema Explorador

Pequenos negocios de alimentacao frequentemente precisam de uma presenca online simples: apresentacao, cardapio, identidade visual e contato rapido. A ideia era testar se um conjunto de templates reaproveitaveis poderia acelerar entregas comerciais.

## O Que Foi Construido

- Tres variacoes visuais publicas.
- Estrutura de temas reaproveitavel.
- Build que injeta o tema escolhido em `public/`.
- Cardapio digital e paginas institucionais.
- Estudo de backend/admin com Express, Prisma e autenticacao.

## Como Apresentar Honestamente

Use este projeto como evidencia de:

- iniciativa comercial;
- capacidade de transformar um nicho em variantes visuais;
- entendimento de deploy estatico;
- evolucao de criterio tecnico ao separar demo, template e SaaS;
- preocupacao posterior com seguranca, documentacao e manutencao.

Evite apresentar como SaaS maduro, produto auditado ou painel pronto para cliente.
