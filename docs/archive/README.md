# Arquivo Historico

Esta pasta guarda documentos antigos, auditorias, planos e rascunhos preservados para contexto historico.

Eles nao representam o posicionamento atual do projeto. A documentacao atual fica em:

- `docs/ARCHITECTURE.md`
- `docs/PORTFOLIO_CASE.md`
- `docs/DEPLOY_NOTES.md`
- `docs/SECURITY_NOTES.md`
- `docs/LIMITATIONS.md`

Se algum arquivo arquivado usar linguagem como "SAFE TO SELL", "pronto para producao" ou notas numericas de seguranca, trate como registro antigo, nao como promessa atual.
