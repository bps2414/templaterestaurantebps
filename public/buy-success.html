Compra Realizada com Sucesso! 🎉

Obrigado pela sua compra! Siga os passos abaixo para colocar seu site de restaurante no ar.

Clone o Repositório

Faça uma cópia do projeto para sua máquina local.

git clone <url-do-repositorio> meu-restaurante
cd meu-restaurante

Configure o Ambiente

Copie o arquivo de exemplo e preencha as variáveis:

cp .env.example .env

Preencha DATABASE_URL, JWT_SECRET e WHATSAPP no arquivo .env

Instale e Rode

Instale as dependências e inicie o servidor:

cd server
npm install
npx prisma migrate dev --name init
npx prisma db seed
npm run dev

Acesse o Painel Admin

Faça login no painel para customizar seu restaurante:

🌐 Site: http://localhost:3000

🔐 Admin: http://localhost:3000/admin

📧 Email: admin@restaurante.com

🔑 Senha: admin123

⚠️ Troque a senha padrão imediatamente após o primeiro login.

Personalize Tudo

No painel admin você pode:

  • ✅ Adicionar/editar pratos com fotos e preços
  • ✅ Criar categorias para o cardápio
  • ✅ Enviar fotos para a galeria
  • ✅ Alterar nome, slogan, horários, endereço
  • ✅ Configurar número do WhatsApp
  • ✅ Alterar links das redes sociais

Deploy (Opcional)

Use Docker para publicar em qualquer servidor:

docker compose up -d

Funciona com Railway, Render, DigitalOcean, AWS, ou qualquer VPS.
